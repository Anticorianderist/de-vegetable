# 下篇：自我教育/3 马克思主义哲学/1 唯物论

在写马克思主义哲学之前，我写一点所谓的阅读须知。

第一，什么是马克思主义哲学？对于国产的大一新生而言，马克思主义基本原理（马原）是必修的思政课之一。和它同系列的课程包括：毛泽东思想和中国特色社会主义理论、中国近代史纲要、思修法基、形势与政策，甚至军事理论等。现在的大学生对这些“红”课刻板印象颇深，大一新生要么极左、要么极右，甚至认为思政课是洗脑课。所以，我回答“什么是马克思主义哲学”主要服务于“它为什么不红”。至少就考研政治的考察范围来看（相信你们思政课的内容也八九不离十），马克思主义基本原理包括：马克思主义哲学、马克思主义政治经济学、科学社会主义。在开始准备考研政治的时候，我以为“背政治”是我负能量的巅峰，因为我对考研政治的刻板印象不弱于任何一个国产大学生。但是，所谓的“洗脑”是无脑的人云亦云。马克思主义基于德国古典哲学、英国古典政治经济学、英法两国的空想社会主义。如果不去“见天地”，你甚至连什么是哲学、什么是政治经济学、什么是空想社会主义都不知道。在展开了马克思主义的三个基本组成部分之后，你至少应该知道马克思主义哲学篇仅仅是发展自德国古典哲学的马克思主义哲学，和马克思主义政治经济学、科学社会主义关系不大。也就是说，马克思主义哲学篇不谈政治，至少比人际关系篇谈得少。

第二，我的哲学水平？为什么不写其它哲学，只写马克思主义哲学？因为我只读过马克思主义哲学。哲学分为很多流派，比如东方哲学和西方哲学，还有什么唯物主义和唯心主义之分，早可以追溯到老子庄子亚里士多德，近一点有马克思主义哲学。我们还可以给哲学附加非常多前缀，比如科技哲学、政治哲学。也许国产大一新生最熟悉的是哲学三问：我是谁？我从哪里来？我到哪里去？和数学篇类似，我只讲我学过的哲学，但仍然和数学篇类似，我可以用三脚猫的哲学水平，呈现哲学足够强的解析力。在本书的框架成型之前，我学过的教材和课程只有：学堂在线的陈为蓬的《逻辑学概论》（因为逻辑学和哲学挂钩，所以勉勉强强也算吧。听完了大部分）；耶鲁大学的《哲学——死亡》（好像只听了三四节课）；肖秀荣的精讲精练（考研辅导书）；徐涛的强化课（考研辅导课）；在本书的框架成型之后，我读完了一本《哲学的慰藉》。也许金庸的《天龙八部》也算，沾一点佛教（其实也算哲学了）。从自知之明的角度来看，真正指导我认识世界的哲学是马克思主义哲学。在开始准备考研政治之后，我发现：牛逼啊！原来马克思主义是这样的！等到你们足够强了，可以读懂我，你会发现包括但不限于我这样的知识分子，乌托邦（社会主义、共产主义）简直成为了一种公知型高级知识分子的执念（高晓松那些废物不算）。和数学篇类似，马克思主义哲学篇是“见天地”的有力武器，又是“见天地”的检验标准。如果可以读懂马克思主义哲学篇，你将有能力触碰最艰深的哲学，因为我足以触碰最艰深的哲学；如果读懂了数学篇和马克思主义哲学篇，你应该已经不需要这本书了。但是，如果连数学篇和马克思主义哲学篇都读不懂，你其实没有什么资格自称“学士”，充其量是一个拿了学位证的专科生而已（可能还不如）。

第三，如何食用哲学？既然我的哲学水平是三脚猫，而且哲学按照流派分类，不像数学是一个大一统的学科。那你能不能食用其它哲学呢？当然可以。比如老子、庄子、王阳明。或者，你就读于某所不上不下的野鸡学校，热衷于你某个另类的思政老师、网红思政老师（比如陈果）口中的哲学。如果你因为或这样或那样的原因不愿意从马克思主义哲学入手，那么你可以跳过本篇，否则我就给你说道说道。马克思主义哲学篇是马克思主义哲学的子集。因此，你需要有一本系统地讲述马克思主义哲学的教材。我比较建议你冲学长学姐要一份盗版或正版的考研政治参考教材，或者直接B站找徐涛的课程直接对着看，只看马克思主义哲学部分。对于马克思主义的历史脉络、逻辑发展脉络、定义，我肯定无法事无巨细地写完。马克思主义哲学篇的特色在于：我会结合一些例子阐释徐涛常用的一个描述“大智慧啊！同学们！”让我们从“哲学 = 世界观 + 方法论”入手。

因为懒得找相关的书籍或教材，我所使用的描述、定义等均出自肖秀荣的精讲精练（2023版）。

## 0x01 哲学基本问题

人生活在世界上，需要对世界有一个总体的看法和把握，这就是人的**世界观**问题。世界观是人们对整个世界的总体看法和根本观点。对此，有一个重要的哲学基本问题区分了唯物主义和唯心主义：物质和意识，谁是世界的本原？

我们抛开哲学，从数学的角度剖析一下这个问题：如何定义物质？如何定义意识？如何定义世界的本原？为什么必须在物质和意识中二选一？比如，物质和意识都是世界的本原、物质和意识都不是世界的本原。

说白了，哲学是一套世界观和方法论。如果你可以提出一套完全自洽的世界观和方法论，比现有的哲学都更加完备、更加自洽，那么你将成为古往今来最伟大的哲学家之一。我们首先预设：哲学家是同时代最具有智慧的一群人，而且他们在前人的基础上迭代，发展出了足够庞杂的哲学体系。因此，我们不得不站在前人的肩膀上。如果你愿意从马克思主义哲学入手，就必须先假设它的某些命题是无需论证的公理，进而去和各种不同的哲学流派辩论。马克思主义是唯物主义哲学。我们终于来到了本文的标题：唯物论。

还有一个重要的哲学基本问题：世界是否可知？

在数学中，我们强调了形式化的重要性。因为没有形式化的框架，也就没有推理所形成的因果链。虽然哲学有一部分形式化的东西（符号学或逻辑学里面的与或非、命题等），但我们所读（看）到的哲学99.9999%都是自然语言哲学，对定义的理解程度也反映了一个人的哲学功底，比如“世界是否可知”的“可知”是什么意思？我发现“苹果A是甜的”。怎么发现的？吃一口就知道了。苹果A当然是世界的组成部分，所以至少“苹果A是不是甜的”是可知的。但是，我们如何判断“苹果B是不是甜的”、“所有苹果都是甜的吗”？

基于此，世界是否可知区分了可知论和不可知论。

- 可知论：认为世界是可以被认识的。
- 不可知论：认为世界是不能被人所认识或不能被完全认识的。

在形式化和推理的层面，数学和哲学可以被挖得非常深，因为它们总是冒出一个又一个未解的命题。在哲学中，某些描述可以被压缩得异常精简。我们用纯哲学的语言压缩一下本节试试：

- 哲学基本问题第一方面：世界的本原是物质还是意识？
  - 唯物主义：物质第一性；意识第二性。
  - 唯心主义：意识第一性；物质第二性。
- 哲学基本问题第二方面：世界是否可知？
  - 可知论：物质和意识具有同一性。
  - 不可知论：物质和意识不具有同一性。

阿巴阿巴阿巴。为什么是第一方面第二方面？第一性、第二性、同一性是啥？这个第一性和马斯克带火的第一性原理有什么关系？而且，我的教材（教辅书）不是这样写的啊！

比如，肖秀荣的精讲精练原句是：

- 唯物主义：把世界的本原归结为物质，主张物质第一性，意识第二性，意识是物质的产物。
- 唯心主义：把世界的本原归结为**精神**，主张意识第一性，物质第二性，物质是意识的产物。
- 可知论：认为世界是可以被认识的，**存在**和**思维**具有同一性。
- 不可知论：认为世界是不能被人所认识或不能被完全认识的，否认**存在**和**思维**的同一性。

我们就顺带谈一下教育基本问题：**这些人抄书都不肯好好抄**。

好的教材浅显易懂，你们可以从知乎搜到不少相关的回答：吐槽国内教材的、推荐国外教材的。总之，我们可以从原句注意到这么一个问题：它们根本不区分同义词用来用去所造成的理解问题。这不仅是文科的问题，而且在理科中也非常严重（比如各种符号体系混乱得不行）。在我所压缩的部分中：物质、存在一律统一为物质；意识、精神、思维一律统一为意识。

有能力必须读经典的原版教材或课程；要选有口碑的译本。提一个醒：不要迷你们特立独行的某个笨比老师，它们自己真的懂哲学和社会科学吗？本科期间，因为我在试验班，没有那种常规的思政课（我只上过思修法基），所以我们的思政课也比较不一般：可能让我们组队做个PPT啥的，还有各种暴论型老师。但是，它们的受众和某些水平极差的网红老师一样多。

回到正题，我不太准确地用大白话概括一下：

- 哲学的第一个基本问题：世界的本原是物质还是意识？
  - 唯物主义：世界的本原是物质。
  - 唯心主义：世界的本原是意识。
- 哲学的第二个基本问题：世界是否可知？
  - 可知论：世界可以被完完全全地认识。
  - 不可知论：世界不可以被完完全全地认识。

马克思主义哲学的立场：唯物主义 + 可知论。

## 0x02 唯物主义

https://www.bilibili.com/video/BV14Y4y1n7hs/

一门哲学得以延续、得以发展、得以让你在21世纪学习，靠的是**它的体系让它的世界观和方法论具有足够的合理性**。举一个例子，亚里士多德。一个受过九年义务教育的高中生，应该可以发现亚里士多德的形而上学错漏百出。当然，我们总不能从一个死了两千多年的哲学家身上找优越感。那么，马克思主义哲学的体系是怎么样的呢？

- **物质**：不依赖于人类的意识而存在，并能为人类的意识所反映的客观存在。
- **意识**：物质世界长期发展的产物，是人脑的机能和属性，是客观世界的主观映像。

我们仿写老子《道德经》的一句话“道生一，一生二，二生三，三生万物”：物质生意识，物质生万物。在老子来看，老子的“道”是世界的本原；在马克思主义哲学看来，世界的本原是物质。至于老子的“三”是哪里得到的，我们就只能将它理解为一拍脑袋了。

首先，马克思主义定义了物质和意识。基于此，才有后面的两个特征、三大规律、五对范畴、认识论、乃至唯物史观、社会主义、共产主义。越根本的定义，越需要长期的、深入的理解。比如老子的“道”，言有尽而意无穷。《道德经》多少字？无论哪个版本，没有超过六千字，但国学大师们、知识分子们、东方哲学家们在六千字不到的《道德经》上雕了多少的花？类似地，我有必要强调：至少对于马克思主义哲学而言，某些定义需要常读常新、直至形成一个足够稳定的理解：物质、意识、实践、主观能动性，否则**基于此世界观的方法论就变成了空中楼阁**。到此，我们也可以顺便解决一个问题：为什么我们“听过很多道理，依然过不好这一生”？有些人怪罪执行力；有些人怪罪意志力；有些人怪罪命运。我问“我们”一句：你世界观的底座在哪里？我们可以继续问那些好为人师的人：你世界观的底座在哪里？我会告诉你：我世界观的底座是马克思主义哲学；至于其ta人，大概率是回答不出来的。

我们姑且认为肖秀荣精讲精练以及我国“罢黜百家，独尊马术”这些教材没有篡改历史：

- 古代朴素唯物主义：物质是金木水火土等具体的物质形态。
- 近代形而上学唯物主义：物质是原子。
- 现代辩证唯物主义：物质是客观实在。

又开始抽象起来了。

大家应该知道所谓的金木水火土，即五行。那时候的物质还沾了太多神佛的元素。当然，如果去追溯唯物主义所有的源头，应该还有其它的定义。但是，后来发展了物理学、化学、生物学，所以有了原子、中子、夸克等微观例子。因此，从古代朴素唯物主义到近代形而上学唯物主义的**基本**发展之一是：对物质的理解从金木水火土等具体的物质形态到原子。那么，近代形而上学唯物主义的优越性在哪呢？随便列举两条：

1. 包含了更多的物质：无论怎么列举金木水火土、苹果A、苹果B、山川湖海、风雨雷电，古代朴素唯物主义都不可能列举出空气，也无法将风解释为流动的空气，但近代形而上学唯物主义一张元素周期表（原子）就可以把万物列举一遍了。
2. 更加大一统：在古代朴素唯物主义中，关于物质的客观规律是本草纲目（某种草药治什么病）；在近代形而上学唯物主义中，关于物质的客观规律是分子（某种生化物质决定什么细胞的什么状态）。

从近代形而上学唯物主义到现代辩证唯物主义就更抽象了。所以，我问两个问题：

1. 电磁场是不是物质？
2. 历史是不是物质？

在近代形而上学唯物主义看来，电磁场当然不是物质，因为它不是原子。既然它不是物质，那是意识吗？假如我是那个时代的唯心主义哲学家，我说：“你看！你们唯物主义根本无法解释电磁场的存在，但我们唯心主义可以！电磁场是上帝所造的，正如一切物质都是上帝所造的。”如果这样的话，唯心主义就压倒了唯物主义。

在近代形而上学唯物主义看来，历史当然不是物质，因为它不是原子。既然它不是物质，那是意识吗？假如我是那个时代的唯物主义哲学家，我说：“历史是一种宏观的意识，是历史所包含的人类的意识有机结合。”如果这样的话，历史是分立的瞬间。我们无法就“历史是分立的瞬间”这种世界观去剖析历史，去提出指导世界的方法论。

但是，在现代辩证唯物主义（马克思主义哲学的立场）看来，电磁场和历史都是物质。我们回到马克思主义哲学对物质和意识的定义：

- **物质**：不依赖于人类的意识而存在，并能为人类的意识所反映的客观存在。
- **意识**：物质世界长期发展的产物，是人脑的机能和属性，是客观世界的主观映像。

电磁场和历史满足现代辩证唯物主义对物质的定义。此外，你可以试试列举出物质所不包的存在（找出马克思主义哲学的bug）。但是，现代辩证唯物主义对于近代形而上学唯物主义的最大改进之二包括：

1. **从静态到动态**：在下一篇文章中，我将讲到马克思主义哲学的两个特征。其中**永恒发展**让马克思主义哲学成为了无敌的存在。从这个角度去看待电磁场，我们可以发现一个有趣的点。电磁场可以由变速**运动**的带电粒子产生。如果读马克思主义哲学，你不难发现时空、运动、动和静是和物质相关的若干重要的定义及分析对象。批评形而上学主打“不思进取”。
2. **唯物史观**：近代形而上学唯物主义在历史上根本打不过同时期的唯心主义，但现代辩证唯物主义可以。

本文不厌其烦地、隆重地对物质和意识展开了讨论。此体系将在《三大规律》、《五对范畴》和《认识论》中爆发出极高的实用性和解析力。
